# textbook
ロボット知能を実際につくっていく上で必要となる、いくつかの項目についてい、説明するpdfファイルやwikiページをここにまとめます。

ロボット知能を研究する上で必要になる知識としては、大雑把に分類すると、
  * 感覚運動写像やニューラルネットワークなど理論的な側面
  * ネットワークやPythonプログラミングなどのソフトウェア的な説明
  * ラズパイやセンサーなど、ハードウェア的な知識
  * Debianを利用するOSに関する知識
などがあります。
